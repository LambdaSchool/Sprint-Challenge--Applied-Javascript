### Questions:
1. What is the DOM?
The DOM is an API that allows JS to alter HTML/CSS.

2. What is an event?
An event is a specific input that occurs and is recognized by the program, such as a mouse click or the page fully loading.

3. What is an event listener?
An event listener is a callback function that makes something happen when it detects the given event. 

4. Why would we convert a NodeList into an Array?
We convert NodeLists to Arrays because NodeLists have few methods, while Arrays have useful ones like .map, .filter, etc.

5. What is a component? 
A component is a piece of code that is modular enough to be reused or slightly edited for a variety of situations, to the point that you can copy paste components from old projects into new ones.
