### Questions:
1. What is the DOM?

Answer: The DOM is the Document Object Model. It is basically the programming interface for HTML documents. It represents a webpage so that programs can change the document structure style and content.

2. What is an event?

Answer: An event is an object that informs the DOM when something has happened, be it a click a mouse scroll or a page loading or whatever.

3. What is an event listener?

Answer: An event listener is a method that gets added a Node Element that listens for when a specified event occurs.

4. Why would we convert a NodeList into an Array?

Answer: Because Nodelists are simply array-like objects and do not necessarily have as much functionality as an Array. A true Array has a ton of methods we can use on it to make it more useful.

5. What is a component? 

Answer: A component is a Javascript class that that you can use to create a dynamic component on a webpage. (Such as a dropdown menu, tabs, etc) The key is that creating a component allows you to easily use that component's functionality many places across your web application.
