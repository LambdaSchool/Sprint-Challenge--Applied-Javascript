### Questions:
1. What is the DOM? The DOM is a copy of all elements the site is composed off, that lies in memory. It is what the browser actually shows (instead of the raw html / css code) and can be manipulated usign JS / jQuery

2. What is an event? An event is a user defined "happening" this could be triggered by a click on an element which then invokes a function or smth alike

3. What is an event listener? A listener is a small method awaiting a specific user action or even (f.e. an error) that then invokes a function

4. What is jQuery? jQuery is a JS framework that was designed to make dom manipulation a lot easier

5. What is a component? A component is a pre-written reusable part of a website / app that has a specific task (like adding functions to all buttons f.e.)
