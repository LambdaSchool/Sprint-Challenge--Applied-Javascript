# Questions

1. What is the DOM?

The DOM is the Document Model Object an API that we use to ineract with our web pages.

2. What is an event?

Events are actions that notify our code of things that have taken place. Thse events can be deteched by our Javascript and the event object that is created gives us infomation about the event that has occured. 

3. What is an event listener?

An event listener is a function that is called whenever a specific event has occured.

4. Why would we convert a NodeList into an Array?

You would convert an NodeList into an Array when you yould need to utilize the methods that provided to Arrays and not NodeLists. 

5. What is a component? 

A componet is a reusable set of logic, behaviors and inferface elements that can be found on our webpages.
